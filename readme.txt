Sticky Genesis Topbar adds an area to the top of your website and lets you customize it from the Genesis Theme Settings page.

== Description ==

> **You must have the [Genesis framework](http://my.studiopress.com/themes/) installed with an active Genesis child theme to use this plugin.**

This plugin adds an area [before Genesis header](https://genesistutorials.com/visual-hook-guide/) and a metabox or section to the Genesis Theme Settings page where users can add or modify text, button text, button URL for Topbar. User also can remove the topbar by checking a checkbox in this section.

= Compatibility =

Sticky Genesis Topbar is compatible with any Genesis child theme, whether created by StudioPress or by a third party developer. 

== Installation ==

1. Upload the entire genesis-topbar folder to the /wp-content/plugins/ directory (or install directly by searching in Plugins > Add New)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the Genesis Theme Setting menu, add or modify content for your topbar from the “Topbar” metabox.

== Frequently Asked Questions ==

= I don’t see any options after activating the plugin. Why not? =

This plugin requires the Genesis framework. If you are using a Genesis theme, navigate to Genesis > Theme Settings to enter your custom topbar text.
